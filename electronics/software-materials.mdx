## Software üíª

### Arduino IDE
![Arduino IDE](https://raw.githubusercontent.com/BenRover-24/docs/main/docs/electronics/images/software/IDE%20arduino.jpeg)

Using the Arduino IDE made programming the Arduino Mega easier.

### KiCad
![KiCad](https://raw.githubusercontent.com/BenRover-24/docs/main/docs/electronics/images/software/Kicad_logo_new.png)

The entry of the electrical diagram and the creation of the printed circuit were carried out on KiCad, which is a suite of free software for drawing electronic diagrams and designing printed circuits, offering a 3D visualization of our PCB and other tools necessary for the design of electronic circuits.

### FlatCAM
![FlatCAM](https://raw.githubusercontent.com/BenRover-24/docs/main/docs/electronics/images/software/flatcam.png)

Flatcam is a software for generating the CNC jobs G-codes.

### Candle
![Candle](https://raw.githubusercontent.com/BenRover-24/docs/main/docs/electronics/images/software/Candle_cnc.png)

Candle is a software for controlling the CNC machine.

### Fritzing
![Fritzing](https://raw.githubusercontent.com/BenRover-24/docs/main/docs/electronics/images/software/fritzing.png)

Using Fritzing, an electronic circuit design software, allowed us to create an overview of the different connections between the project components.
